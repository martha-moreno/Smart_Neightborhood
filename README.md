# Smart Neighborhood Management System

A web-based application designed to improve communication, security, and management within a smart neighborhood. This system enables residents and administrators to efficiently manage community activities, monitor services, and enhance overall living experience.

---

## Project Overview

The **Smart Neighborhood Management System** is built using **ASP.NET** and provides a centralized platform for:

* Resident management
* Community communication
* Service tracking
* Secure access and authentication

This project was developed as part of an academic team assignment.

---

## Features

* 👤 User Registration & Login (Secure authentication)
* 🏠 Resident Management
* 📢 Community Announcements
* 🛠️ Service Requests / Tracking
* 🔐 Password Encryption (using BCrypt)
* 🌐 Responsive UI (Bootstrap)

##  Setup & Installation

### 1. Prerequisites

* Install **Visual Studio**
* Install **.NET Framework (compatible version)**
* Enable **ASP.NET Web Development** workload

---

### 2. Run the Project

1. Extract the ZIP file
2. Open:

```id="open1"
Smart_Neightborhood_Project5_Team19.sln
```

in Visual Studio

3. Restore NuGet packages (automatic or right-click solution → Restore)

4. Click **Run (▶)** or press:

```id="run1"
Ctrl + F5
```
